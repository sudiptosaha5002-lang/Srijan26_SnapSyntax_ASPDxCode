# Srijan26_SnapSyntax_ASPDxCode

Truck Logistics is a modern logistics platform built with React and Vite. It combines truck booking, shipment tracking, operations dashboards, and a cinematic simulation module into one presentation-ready product.

## Project Highlights

- Truck booking flow with live quote behavior
- Logistics tracking desk with shipment history
- Operations pages for dashboard, orders, payments, requests, analysis, and settings
- 3D-style simulator experience with truck showcase and route scene
- Team-based About page with custom crew cards
- Dark and light theme support
- Responsive UI across landing, booking, tracking, operations, simulator, About, and Contact pages

## Tech Stack

- React
- Vite
- Framer Motion
- React Router
- Leaflet / React Leaflet
- Three.js / React Three Fiber / Drei

## Routes

- `/` Home
- `/booking` Truck booking
- `/tracking` Logistics tracking desk
- `/dashboard` Simulation system
- `/ops/dashboard` Operations dashboard
- `/ops/orders` Orders
- `/ops/payments` Payments
- `/ops/requests` Requests
- `/ops/analysis` Analysis
- `/ops/settings` Settings
- `/about` About page
- `/contact` Contact page

## Getting Started

```powershell
cd E:\TruckLogistic
npm install
npm run dev
```

Open the local Vite URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Build
